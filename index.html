Welcome to BagaWork!

The perfect framework for app beginners.

What's this?

BagaWork is a JavaScript framework one can use to build smarthpone apps. It has been designed to be very easy to learn and use for those that have never programmed before, but it still has the abilty to create quite cool and useful apps in it. It is also very easy to share the apps you create with your friends, and to install them on different type of devices that support installing Progressive Web Applications, such as most Android smartphones.

Why this?

Many people trying to learn programming quickly give up because they think programming is too hard. By instead learning programming through BagaWork, they will hopefully keep on fighting and keep learning more and more about programming since they find it more fun when they can create useful apps almost directly from the start, and easily share and install them on different devices.

How to create an app?

We have developed an Online Editor you can use to create your BagaWork apps in. However, instead of using the online editor on this website (the documentation website for BagaWork), you probably want to create an account at our BagaLand website, where you can also save your apps online and let other users install them on their devices.

How to install an app?

BagaWork apps on this website can't be installed, but BagaWork apps hosted on BagaLand can, so head over there if you want to try installing an app. And if you want to make your own apps installable, create and store them there.
